# Noctrinth

**Noctrinth** is a fork of [Modrinth](https://github.com/modrinth/code). This monorepo includes the desktop app, web frontend, and shared packages.

For documentation, architecture, and general infrastructure details refer to the **[original repository](https://github.com/modrinth/code)** — everything is covered there in depth.

---

### New features in the desktop app
- **Collections** — full section: browse, create, edit, delete, add/remove projects
- **Followed projects** — a virtual collection of all projects the user follows
- **Notifications** — Modrinth notifications page built into the app
- **"Save to collection" button** on project pages
- Modrinth OAuth flow reworked: replaced the embedded WebView
- Modal overlay — changed from green to soft purple
